#### GitHub Blog (AI)
- **核心论点**：GitHub 与 Microsoft 合作，通过应用基于 LLM 的上下文验证来减少秘密扫描中的误报。
- **关键要点**：
  - 秘密扫描对于保护开发者和组织免受暴露凭证的风险至关重要。
  - 通过与 Microsoft 合作，GitHub 引入了更多的上下文推理到秘密扫描验证中。
  - 该方法通过查看检测到的值在代码中的使用情况来区分真实暴露和仅看起来敏感的值。
  - 该方法在数百个客户确认的误报警报上进行了评估，目标是减少 65%，实际结果是 75.76%。
  - 减少误报意味着更少的噪音和更高比例需要操作的警报。
- **值不值得深读**：这篇文章详细介绍了 GitHub 如何通过技术创新提高安全系统的效率和准确性，对于安全领域的专业人士和开发者来说非常有价值。
