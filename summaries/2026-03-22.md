## 今日要点

**1. Jas Sekhon 出任 Google DeepMind 首席战略官**

政治学家、数据科学家 Jas Sekhon 宣布加入 Google DeepMind，担任 Chief Strategy Officer，直接向 Demis Hassabis 汇报。他明确表示此举出于道德责任感——「responsibly shepherd AGI into the world, benefiting everyone」。一位非传统 AI 技术背景人士进入顶级 AI 实验室的战略核心位置，折射出 DeepMind 在 AGI 阶段对政策、伦理与社会影响能力的重视程度正在上升。

**2. Paul Graham 发布长文《The Brand Age》：技术抹平差异后，品牌成为最后的护城河**

PG 以瑞士钟表业为案例，论证了一个对科技创业者极具警示意味的规律：当技术进步将产品间的实质性差异消除后，品牌成为唯一剩余的竞争壁垒。这个逻辑对当下 AI 时代尤其值得警惕——当 LLM 能力趋于同质化，谁在建立品牌，谁在打价格战，结局将截然不同。

**3. OpenAI 联手 Amazon Bedrock 推出 Stateful Runtime Environment for Agents**

OpenAI 正式介绍了在 Amazon Bedrock 上为 agent 提供有状态运行环境的能力。这意味着 AI agent 在执行长任务时可以保持上下文状态，是 agent 从「单次对话」走向「持续工作流」的关键基础设施升级，对企业级 agent 部署有直接影响。

**4. Claude Code CLI 获得开发者强力背书，AI 编程工具渗透深度持续加深**

Twitter 上多位开发者对 Claude Code 给出极高评价，有用户表示「启动一个新项目全程没有手动操作过 Chrome，Claude Code 包揽一切」。与此同时，Browser Use CLI 2.0 发布，号称速度提升 2 倍、成本减半，AI 驱动的浏览器自动化工具正在快速成熟。

**5. OpenAI 向美加大学生提供 $100 Codex 额度，加速教育市场渗透**

OpenAI 总裁 Greg Brockman 亲自宣布向美国和加拿大在校大学生提供 $100 Codex credits。这是 OpenAI 在高校市场的定向补贴动作，显示其在争夺下一代开发者用户习惯上的主动出击。

### Paul Graham Essays

**[The Brand Age](https://paulgraham.com/brandage.html)**

**核心论点**：技术进步天然地消除产品间的实质差异，而差异消失后，品牌成为唯一剩余的护城河——这不是偶发现象，而是我们时代最强大的结构性力量之一。

**关键要点**：
- 瑞士钟表业在1970年代遭遇三重打击：日本竞争、布雷顿森林体系崩溃导致瑞士法郎大幅升值、石英运动的出现，单位销量跌去三分之二
- 活下来的品牌（如 Patek Philippe）选择放弃「性能竞争」，转型为奢侈品牌；Omega 执迷于技术迭代，最终破产
- 黄金时代（1945-1970）的钟表追求「薄」与「准」两个实质性维度；今天贵的表贵在营销和人为稀缺，而非工程
- 品牌转型耗费了瑞士钟表业约20年时间，完成后营收反而比黄金时代更高
- 「Brand is what's left when the substantive differences between products disappear.」
- 对 AI 时代的隐喻极强：当 LLM 能力同质化，率先建立品牌心智的公司将获得不成比例的回报

**值不值得深读**：强烈推荐，这是 PG 近年难得的长篇产业分析，对任何正在思考 AI 时代竞争格局的创业者或投资人都有直接的战略参考价值。完整原文信息量远超摘要。
