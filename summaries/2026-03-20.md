## 今日要点

**1. Subagents 模式正成为编程 agent 的标准范式**
OpenAI Codex、Claude Code、Gemini CLI、Mistral Vibe 等主流编程 agent 均已正式支持 subagents（子代理）功能。Simon Willison 专门为此写了系统性指南，Josh Cohenzadeh 在 SF meetup 的调查也显示 Claude Code 以 90% 使用率遥遥领先。这标志着 AI 编程工具从「对话辅助」向「多代理并行执行」的架构升级，是今年最重要的工程模式转变之一。

**2. Anthropic 同步发力生态与研究：一周内推出研究院、1 亿美元合作伙伴网络、81,000 人 AI 调研**
Anthropic Institute（研究院）、Claude Partner Network（1 亿美元投资）和大规模用户 AI 需求调研三件事集中发布，表明 Anthropic 在技术之外开始系统性建设教育、商业和政策影响力，战略重心正从「做好模型」向「定义行业规则」迁移。

**3. Google AI Studio 上线全栈 vibe coding 体验，整合 Antigravity + Firebase**
Google 将 AI Studio、Antigravity coding agent 和 Firebase 打通，支持直接生成具备后端能力的多人实时应用。这是 Google 在「一站式 AI 开发环境」上的重大产品动作，直接对标 Cursor 和 Replit，值得密切关注其对开发者生态的冲击。

**4. Physical Intelligence 发布快速 RL 微调方法，数小时内完成精细任务适配**
π0.6 模型通过新增「RL token」输出头，配合轻量 actor/critic 网络，无需重训全模型即可在数小时甚至数分钟内完成针对精细任务的强化学习微调。这对机器人落地部署意义重大——将定制周期从「数周」压缩到「数小时」。

**5. Mistral Small 4 发布：119B MoE 模型统一推理、多模态与 agent 编程能力**
Apache 2.0 开源许可，整合了 Magistral（推理）、Pixtral（多模态）、Devstral（agent 编程）三大旗舰能力于一身，支持可调节的 reasoning_effort 参数。这是迄今为止单一开源模型能力集成度最高的版本之一，对于不想在多个模型之间切换的开发者具有很高吸引力。

### Simon Willison's Blog

**《Subagents》**

Simon Willison 是知名的 AI 工具独立研究者和 Django 核心开发者，这篇是他「Agentic Engineering Patterns」系列指南中的 Subagents 章节，是目前关于 subagents 模式最系统完整的开发者参考文档。

**核心观点：subagent 模式的本质价值不是「更多并发」，而是「保护根节点的 context 预算」。** 每次派发 subagent，相当于用一个新鲜的 context window 去完成一个定界任务，避免把昂贵的顶层 context 浪费在大量探索性工作上。

最有实操价值的洞见包括：Claude Code 的 Explore subagent 会在每次新任务开始时自动扫描代码库结构，返回父 agent 所需的精炼摘要；parallel subagents 可以同时运行，甚至可以用更便宜的 Claude Haiku 加速；specialist subagents 可以被赋予 code reviewer、test runner、debugger 等专属角色和系统提示。

原文中最有力的一句话来自对 Claude Code 探索 subagent 的观察：*"It's interesting to see models prompt themselves in this way - they generally have good taste in prompting strategies."*

原文链接：https://simonwillison.net/guides/agentic-engineering-patterns/subagents/#atom-everything
